Spanish Verb Conjugator

Instructions

  • In the text box below, write a
  • The API will accept two different requests:
    1. /{verb}/{tense}/{mood}/{subject}
    2. /{verb}/imperative/{subject}
    This is because the imperative mood does not really associate with a tense. If your request is not of one of these two forms you will receive a 404 error.
  • You can use any verb in the Spanish language as long as it ends in -ar, -er, or -ir. Do not write the 'se' for reflexive verbs, just leave it off.
  • The moods and their tenses are:
    1. Indicative: conditional, conditionalperfect, conditionalprogressive, future, futureperfect, futureprogressive, imperfect, nearfuture, pastprogressive, pluperfect, present, presentperfect, preterite.
    2. Subjunctive: imperfect, pluperfect, present, presentperfect.
    3. Imperative
  • For indicative and subjunctive moods, the subjects are: yo, tu, usted, nosotros, vosotros, ustedes.
  • For the imperative mood, the subjects are: tu, vosotros, usted, ustedes, and nosotros.
